Dominio y Hosting

Dominio
aisolutionsperu.com — Dominio .com registrado en un proveedor de nivel internacional (Namecheap o GoDaddy). El dominio .com fue elegido por ser el más reconocido a nivel global y transmitir credibilidad empresarial.
Hosting
Servidor de hosting compartido para la fase inicial. Es suficiente para una web informativa estática con formularios y contenido multimedia. Proveedores recomendados: SiteGround, Hostinger o Bluehost. Cuando el tráfico crezca, se puede migrar a un VPS (servidor virtual privado).
Tipo de sitio
Sitio web estático desarrollado íntegramente con HTML5 y CSS3. Sin uso de frameworks de JavaScript ni bases de datos en esta versión.
Accesibilidad
El sitio cumple con las pautas WCAG 2.1 de nivel AA: atributos lang, alt en imágenes, etiquetas label en formularios, jerarquía de encabezados y contraste de colores.
Compatibilidad de navegadores
Compatible con Google Chrome, Mozilla Firefox, Microsoft Edge, Safari y Opera. Diseño responsive para dispositivos móviles, tablets y escritorio.
